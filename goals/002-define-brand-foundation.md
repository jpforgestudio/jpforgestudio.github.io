# Problem

현재 브랜딩은 깔끔하지만 약간 익명적이다.

JP Forge Studio와 SaneFocus의 관계가 시각적으로 선명하지 않고, favicon도
스튜디오 이니셜처럼 읽혀서 SaneFocus의 핵심 경험인 "작은 노트 창을 열고,
생각을 적고, 원래 앱으로 돌아간다"는 느낌이 약하다.

# Goal

브랜드 구조를 가볍게 정리하고, 사이트에 즉시 보이는 작은 브랜드 시그널을
추가한다.

Primary brand:

- SaneFocus

Endorser / maker brand:

- JP Forge Studio

Visual motif:

- small floating note window
- text caret
- focus return / app window memory
- quiet Mac utility surface

Voice:

- short
- concrete
- calm
- native
- local-first

Avoid:

- productivity hype
- SaaS platform language
- abstract monograms that do not connect to the product
- decorative brand marks that overpower the app

# Plan

Keep this pass small.

Primary changes:

- Replace the favicon with a compact focus-window mark.
- Make the header brand mark echo the same focus-window motif.
- Adjust homepage copy so JP Forge Studio points more clearly toward SaneFocus.
- Keep support/privacy copy unchanged unless brand clarity requires it.

Checks:

- Favicon loads as SVG.
- Header mark remains legible at small size.
- Homepage still feels like a quiet Mac utility site.
- SaneFocus remains the product visitors remember.
